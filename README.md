# Premier League Data Viewer

This is a Next.js application to display Premier League data from 1888/1989 to 2023/2024.

## Features

*   **Dark Theme:** A clean and modern dark theme inspired by FotMob.
*   **Interactive Navigation:** A professional and easy-to-use menu navigation bar with a scrollable seasons dropdown.
*   **Stylistic Content Boxes:** Information is presented in clear, well-defined content boxes.
*   **Detailed Standings:** View Premier League standings for each season, with the winner's name bolded.
*   **Soccer-Themed Background:** A subtle soccer field background image is displayed on all pages.

## Getting Started

To run the website, follow these steps:

1.  Install the dependencies:

    ```bash
    npm install
    ```

2.  Run the development server:

    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Gemini API Key

This project uses the Gemini API. The API key is stored in the `gemini.md` file. Please do not commit this file to any public repository.

## Data

The data is provided in CSV format in the `/data` directory.
